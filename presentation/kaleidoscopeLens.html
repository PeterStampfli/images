Anti-aliasing

Pixels have a finite size !

Circle inversion expands pixel sizes. Mapped output image pixels may contain many input image pixels.

Taking the color of only one input image pixel then results in image noise.

Use instead an average of the color inside a region of the size of the mapped output image pixel.
This is fast using summed area tables for rgb color components.
Strongly improves the output image. Even far away from the border of a Poincar√© disc.
