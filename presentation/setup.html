setup

Three transforms give the color of an output image pixel with indices (i,j):

A linear transform to points in geometrical space
(x,y) = scale * (i,j) + offset.

The nonlinear mapping of (x,y)
creates the symmetries.

a linear transform to indices (h,k)
of input image pixels
(h,k)=rotation*scale*(x,y)+offset.

The color of a pixel at (i,j) results from the input image at (h,k).
To get good results we need anti-aliasing and interpolation.

We can directly interact with these transformations!
