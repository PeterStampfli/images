Interpolation

The mapping shrinks pixels for elliptic geometry. Many output image pixels go to the same input image pixel.

Without color interpolation they all get the same color and the ouput image has big pixel blocks.

Bi-linear interpolation gives much better results. Blocks disappear.
Linear artefacts appear at maxima and minima of luminosity.

Bi-cubic interpolation eliminates these linear artefacts. Mitchell-Netravali with a separable kernel is very fast. Only the hard mirror lines make some "artificial" lines.
