Interaction with images:
Mouse indicator changes to "crosshair" or "pointer". Click, drag and turn the mouse wheel.

Keyboard shortcuts:
"f" and "F" - change font size
"n" and "N" - go to next/previous page
"z" and "Z" - zoom in/out
"i" - use another input image
